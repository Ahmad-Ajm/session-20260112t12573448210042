# Specify

## Functional Requirements
- User can create a CV with personal info, education, and experience.
- User can upload a profile image (max 2MB).
- User can edit and save CV changes.

## Validation Rules
- Required fields: name, experience, education.
- Image must be jpg or png and <= 2MB.

## Data Fields
- Full Name
- Phone Number
- Education
- Experience
- Profile Image (optional)
